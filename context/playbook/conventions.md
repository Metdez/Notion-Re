## 5. Icon convention
- Every new page/record gets a **Notion icon** emoji.
- Pick an icon that reads at a glance (e.g. company → building/briefcase; person → user).
- Record the chosen icon convention here once settled so it stays consistent.


## 6. Sourcing format
- Every factual field includes the **source URL inline**, hyperlinked on the data itself.
- No source → don't add the fact.
- Example: `Series B — $40M (Jun 2025)` with the funding figure linked to the announcement.
